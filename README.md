# ControleDeBar

![](.gif)

## Introdução 
 
 Trata-se de um programa criado para o controle de bares de forma simples e prática,
 possibiilitando o cadastro de mesas, garçons, produtos e abertura/fechamento de contas
 classe que envolve todas as outras para chegar a um resultado do valor que deverá ser 
 pago.  
 Outra funcionalidade implementada é o filtro de contas por aberta/fechada.
 
 ## Funcionalidades
 
 - O usuário pode criar, editar, visualizar e excluir mesas, garçons, produtos e mesas.
 - Os principais métodos utilizados na implementação do programa foram:

 > Orientação a Objetos: Facilitando a compreenção e edição do mesmo.
 
 > Encapsulamento: Protege os dados que estão sendo utilizados no programa.

 > Herança e Polimorfismo: Reutilização do código, facilitando a compreenção e futuras edições.
 
 > Utilização do framework ASP.NET MVC, separando o código e facilitando a compreensão,
 edição e implementação no código.

 > Armazenamento das informações inseridas/editadas em arquivo json em maquina.

 > Minima estilização com BootStrap para facilitar a compreensão e visualização do site.
